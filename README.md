# Skip List Implementation in C++ 🚀

A probabilistic data structure implementation based on **William Pugh's 1990 paper**, designed to achieve $O(\log N)$ search complexity without the need for complex tree rebalancing.

### 📊 Key Features
- **Dynamic Level Generation:** Uses a randomized coin-flip logic (`p=0.5`) to build layers.
- **Performance Analysis:** Includes scripts to generate Time vs. Input Size plots verifying logarithmic growth.

### 📈 Benchmarking
This project benchmarks Search, Insertion, and Deletion operations, proving that performance scales logarithmically as $N$ increases, offering a significant advantage over linear structures.
